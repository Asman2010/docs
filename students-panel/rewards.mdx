## Checking Available Rewards

The rewards system allows you to view and redeem prizes based on your earned points.

### Viewing Rewards

Click the "Check Rewards" button to view a table of all available rewards, including:
- Reward name
- Points required
- Reward description
- Availability status

<img src="/images/student-rewards.png" alt="Rewards Table" />

### Reward Table Details

| Column | Description |
|--------|-------------|
| Reward | Name of the available reward |
| Points | Points required to redeem |
| Description | Details about the reward |
| Status | Whether the reward is currently available |
