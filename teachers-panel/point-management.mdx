## Managing Student Points

The point management system allows teachers to modify student points through a simple interface.

### Steps to Modify Points

1. Select the class from the "Choose by class" dropdown
2. Click either the "Increase" or "Decrease" button
3. Select the point value from the dropdown menu
4. Click "Submit" to apply the changes

<img src="/images/checks-passed.png" alt="Point Management Interface" />

### Confirmation

After submitting points, you'll receive a confirmation dialog indicating successful submission.

<Note>
  Points are immediately reflected in the student's record after submission.
</Note>
