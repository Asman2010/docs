## Getting Started

The teacher's panel provides a comprehensive interface for managing student points and generating reports. When you first access the panel, you'll see a clean interface with a dropdown menu labeled "Choose by class".

<img src="/images/checks-passed.png" alt="Teacher's Panel Overview" />

### Key Features

- **Class Selection**: Choose specific classes to view enrolled students
- **Point Management**: Increase or decrease points for individual students
- **Report Generation**: Create comprehensive student reports
- **Feedback System**: Submit bugs, feedback, or feature requests
