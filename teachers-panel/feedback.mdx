## Feedback Submission

The feedback system allows teachers to communicate issues or suggestions to the development team.

### Submission Types

- Bug Reports
- General Feedback
- Feature Requests

### How to Submit

1. Select the feedback type from the dropdown
2. Provide a descriptive title
3. Add detailed description
4. Click submit
